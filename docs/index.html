brute(transitions)

This function generates a test plan such that all possible model states and transitions from those states are exercised.

Parameters

  • Array transitions :

    a list of transition objects.

Returns

Array :

a test plan.

quick(transitions, options)

This function generates a test plan with a single test which is the result of a random walk through the test models transitions.

Parameters

  • Array transitions :

    a list of transition objects.

  • Object options :

    a set of options for the strategy.

    • number options.seed

      a seed value for the PRNG driving the walk.

Returns

Array :

a test plan.
